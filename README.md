# TaskFlow Internship Project

TaskFlow is a full-stack task management system developed during my internship to practice backend and frontend development.

The application allows users to authenticate, manage projects, and manage tasks through a secure REST API.

---

## Technologies Used

### Backend
- Node.js
- Express.js
- MySQL
- mysql2
- JWT Authentication
- bcryptjs
- dotenv
- CORS

### Frontend
- React.js
- React Router
- Axios
- Vite

### Tools
- Git & GitHub
- Visual Studio Code
- MySQL

---

## Features

### Authentication
- User login system
- Password verification using bcrypt
- JWT token authentication
- Protected routes
- Automatic redirection for unauthorized users

### Project Management
Users can:
- View projects
- Create projects
- Update projects
- Delete projects

### Task Management
Users can:
- View tasks
- Create tasks
- Update tasks
- Delete tasks
- Assign tasks to projects

---

## Backend Security

The backend includes:

- Environment variables for sensitive information
- Protected API routes using JWT
- Secure password comparison
- User identification through JWT tokens
- Database credentials stored outside the source code
